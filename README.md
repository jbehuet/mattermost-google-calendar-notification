# MATTERMOST GOOGLE CALENDAR NOTIFICATION

### Prerequisites
* [NodeJS](https://nodejs.org/en/)

### Installation

```bash
git clone https://github.com/jbehuet/mattermost-google-calendar-notification.git   
cd mattermost-google-calendar-notification
npm i
```

### Production

`npm start`
